sofia pro
Sofia Pro is a sans font family origianlly created in 2009 by Olivier Gourvat from Mostardesign. Later redesigned in 2012, this geometric font has recieved approval from many graphic industry professionals. Sofia Pro is included in a set of 16 font styles, which allows it to compliment any type-based project .
